Tentang Saya

Lulusan S1 Teknik Elektro dengan motivasi tinggi untuk mengembangkan kompetensi di bidang elektronika, pemrograman mikrokontroler, firmware, serta desain perangkat keras. Terbiasa mengerjakan proyek akademik, melakukan penelitian, serta bekerja sama dalam tim. Memiliki pemahaman dasar mengenai pemrograman C/C++, Python, komunikasi data mikrokontroler, dan desain PCB.

Pengalaman Proyek

Sistem Kendali Motor BLDC dengan IMU

Mengembangkan sistem kendali motor BLDC menggunakan algoritma PID dan sensor IMU untuk aplikasi stabilisasi gimbal. Implementasi menggunakan ESP32 dengan komunikasi serial untuk monitoring real-time.

ESP32 C++ PID Control IMU

IoT Monitoring System

Merancang sistem monitoring berbasis IoT untuk pemantauan parameter lingkungan menggunakan berbagai sensor dan ESP32. Data ditampilkan pada dashboard web dengan update real-time.

ESP32 IoT Web Server Sensors

Desain PCB Custom Controller

Mendesain dan membuat PCB custom untuk kontroler mikrokontroler dengan berbagai modul komunikasi dan interface sensor. Meliputi pembuatan skematik, routing, dan testing prototipe.

PCB Design AutoCAD Electronics Testing
